This is an ATM program built with the Java Swing framework. 
It can perform deposits, withdrawals, transfers betwen two account types
(checking and savings) and balance checking functions. 
For testing purposes, both accounts start with 100$ in them. 

1. You can use the ATM.jar file to run the program. 
2. The source code is contained with the src folder, ATM/src/ATM.java
